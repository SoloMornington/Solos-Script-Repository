Solo's Script Repository by SoloMornington

A collection of well-commented LSL scripts.

This script repository will hopefully illustrate some best practices for LSL scripting, and will give both novice and advanced users something to learn from. Emphasis is on well-commented, easily-readable code that can be easily modified.

Dependencies

These scripts should run on Second Life. Many will work on OpenSimulator sims as well.

Install

Download the package. Use the scripts. :-)
